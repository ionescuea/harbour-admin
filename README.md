# Harbour Admin

Static website for **Harbour Admin** — fixed-scope back-office support for Plymouth landlords and micro-charities.

## Pages

| File | Purpose |
|------|---------|
| `index.html` | Home |
| `services.html` | Services and how it works |
| `packages.html` | Monthly packages and fixed-price projects |
| `why-harbour.html` | Approach and who it is for |
| `contact.html` | Enquiries |

## Assets

- `styles.css` — shared layout and design
- `icons.svg` — inline SVG icons
- `images/logo.png` — logo (navbar and favicon)
- `images/logo-hai.png` — alternate logo asset

## Local preview

Open `index.html` in a browser, or serve the folder with any static file server. No build step required.

## Deployment

This is a plain HTML/CSS site. Deploy the repository root to any static host (for example [Netlify](https://www.netlify.com/) or [GitHub Pages](https://pages.github.com/)).

- **Build command:** none  
- **Publish directory:** `/` (root)

After connecting the repo to your host, updates pushed to `main` can redeploy automatically.
